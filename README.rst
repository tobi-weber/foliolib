=======
PyOkapi
=======

PyOkapi is a manager and library for `Okapi <https://github.com/folio-org/okapi>`_ 
and `Folio <https://www.folio.org/>`_.
The Library is automatically generated from the RAML files of the different Folio-Modules.

Documentation can be found `here <https://>`_.


Installation
============

The installation requires python 3.6 or higher.

.. code-block:: bash

    $


Command-line interfaces
#######################

Pyokapi provides two command-line interfaces.

.. code-block:: bash

    okapicli --help

After the first call of okapicli, pyokapi creates a directory with configurations
under $HOME/.pyokapi.


.. code-block:: bash

    foliocli --help
